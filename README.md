# Software Developer Portfolio Template

A clean, responsive portfolio template for software developer students. Built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Sections for About, Skills, Projects, Education, and Contact
- Modern, professional styling
- Easy to customize

## Customization

1. Replace placeholder content in `index.html` with your personal information
2. Update the skills list to reflect your expertise
3. Add your project details and links
4. Modify colors and fonts in `css/styles.css` if desired
5. Update contact information

## Deployment

This is set up for GitHub Pages. Simply push to your `main` branch and enable Pages in repository settings.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
